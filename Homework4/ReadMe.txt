To Run the code follow the instruction mentioned below:

Step 1: Unzip the project folder in your homedrive.
Step 2: Import the project and Build it in your visual studio.
Step 3: For selecting model type the model number and enter.
Step 4: Other functions can be executed using keys from 1-6
Key "1": Color coding using unique polygon ID.
key "2": Color coding based on normal
key "3": Color coding using the formula mentioned in the report
key "4": Bounding box according to second moment
key "5": Bounding box according to first moment
key "6": Bounding box according to second moment using Normals.
